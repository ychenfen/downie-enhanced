核心功能

完整迁移 Downie 4 的所有核心功能，并采用现代 Web 技术栈重构

多平台支持

支持 YouTube、小鹅通、M3U8 流媒体等多种视频平台，智能识别视频源

实时进度

WebSocket 实时连接，实时显示下载进度、速度和剩余时间

Chrome/Firefox 扩展，右键即可下载，无缝集成浏览体验

FFmpeg 处理

内置 FFmpeg 支持，自动转码、格式转换和质量优化

本地部署步骤

1

克隆仓库

git clone https://github.com/ychenfen/downie-enhanced.git
cd downie-enhanced
2

安装依赖

# 后端依赖
cd backend && pip install -r requirements.txt

# 前端依赖
cd ../frontend && npm install
3

启动服务

# 启动后端 (终端1)
cd backend && python main.py

# 启动前端 (终端2)
cd frontend && npm start

启动后访问 http://localhost:3000 即可使用

Docker 部署步骤

1

克隆仓库

git clone https://github.com/ychenfen/downie-enhanced.git
cd downie-enhanced
2

一键启动

docker-compose up -d

Docker 会自动处理所有依赖，访问 http://localhost:3000
